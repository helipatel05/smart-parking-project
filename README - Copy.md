# Smart Parking System

## Description

Smart Parking System is a fully static web app for city-based mall and PVR parking reservations. Users can select a city and parking location, view live slot availability, pre-book a slot, simulate AI number plate gate verification, check predictions, view timers, get indoor navigation steps, and monitor admin statistics.

## How to Run

Open `index.html` directly in any browser.

For local preview, you can also run a simple static server:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Deployment

This project is ready for GitHub Pages, Netlify, or any static hosting service.

Upload these files and folders:

- `index.html`
- `login.html`
- `style.css`
- `script.js`
- `assets/`

No backend server or database is required.
